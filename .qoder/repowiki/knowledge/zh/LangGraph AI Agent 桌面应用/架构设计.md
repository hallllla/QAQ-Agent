- 采用 Electron 标准三层架构：主进程 (`src/main.ts`) 负责窗口管理与 IPC 通信，预加载脚本 (`src/preload.ts`) 通过 `contextBridge` 暴露安全 API，渲染进程 (`src/renderer/`) 基于 React 实现 UI。
- 核心逻辑位于 `src/agent.ts`，利用 `@langchain/langgraph` 的 `StateGraph` 构建智能体工作流，集成计算器、时间查询等自定义工具，并支持 OpenAI 与 Ollama 模型切换。
- 配置管理由主进程通过文件系统持久化存储 (`agent-settings.json`)，并通过 IPC 通道与前端设置面板同步。
- 构建体系基于 `electron-forge` 与 `vite`，通过 `forge.config.js` 协调主进程、预加载脚本及渲染进程的编译与打包流程。
